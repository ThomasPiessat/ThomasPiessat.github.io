# Npm command

## Initialization: 
Start a new project
 ```sh
 npm init
 ```
It will create a ```package.json``` file.

## Install Dependencies: 
Easily add and manage project dependencies
```sh
npm install
```

## The ```i``` flag is an alias for ```install```:
```sh
npm i package
```

## Install latest:
```sh
npm i <dependencie>@latest
```
