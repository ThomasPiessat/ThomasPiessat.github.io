# Visual Studio Shortcuts
## Find in the solution explorer where is the current open file
```ps1
CTRL + ) + S
```

## Search for a line (by number)
```ps1
CTRL + G
```

## List of copy/past
```ps1
CTRL + SHIFT + G
```

## Search for something
```ps1
CTRL + SHIFT + F
```

## Search for a file name
```ps1
CTRL + SHIFT + T
```

## Duplicates a line
```ps1
CTRL + E + V
```
