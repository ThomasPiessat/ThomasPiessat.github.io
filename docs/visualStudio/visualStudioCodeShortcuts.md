# Visual Studio Code Shortcuts
## The code formatting is available in Visual Studio Code through the following shortcuts:
```ps1
Shift + Alt + F
```

## Move Line up
```ps1
Alt + up Arrow
```

## Copy line up/down
```ps1
Shift + Alt + arrow down 
```
