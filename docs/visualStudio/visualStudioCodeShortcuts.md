# Visual Studio Code Shortcuts
## Code formatting
```ps1
Shift + Alt + F
```

## Opens up command palette
```ps1
Ctrl + Shift + P
```
 
## Move Line up
```ps1
Alt + up Arrow
```

## Copy line up/down
```ps1
Shift + Alt + arrow down 
```
